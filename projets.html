Bot Discord et son site web

CatBot est un bot Discord conçu pour améliorer la gestion et l'animation des serveurs.

  • Modération : commandes pour bannir, expulser, gérer les messages et mettre en sourdine les utilisateurs.
  • Gestion : système de tickets et messages de bienvenue personnalisés.
  • Commandes interactives : création d'équipes aléatoires et fonctionnalités musicales.

Pour plus d'informations, visitez le site officiel : catbot.belucraft.fr.

Bot musique pour Discord

Code d'un bot discord open source. Il permet ainsi de jouer de la musique dans un salon vocal, de gérer les files d'attente.

  • Lecture de musique : possibilité de lire des musiques depuis YouTube, Spotify, etc.
  • Gestion des files d'attente : ajout, suppression et réorganisation des chansons.
  • Commandes de lecture : lecture, saut de chansons.

Pour plus d'informations, visitez le dépôt GitHub : github.com/N2630/Music-Bot.
