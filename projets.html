Bot Discord et son site web

CatBot est un bot Discord conçu pour faciliter la gestion et animer les serveurs. Voici ses fonctionnalités principales :

  • Modération : commandes pour bannir, expulser, gérer les messages et mettre en sourdine.
  • Gestion : système de tickets et messages de bienvenue personnalisés.
  • Commandes interactives : gestion d'équipes aléatoires et fonctionnalités musicales.

Pour en savoir plus, visitez le site officiel : catbot.belucraft.fr.

Bot musique pour Discord

Ce bot permet de diffuser de la musique dans un salon vocal et de gérer des playlists. Ses fonctionnalités incluent :

  • Lecture de musique : lecture de morceaux depuis YouTube, Spotify, etc.
  • Gestion des playlists : ajout, suppression et réorganisation des chansons.
  • Commandes intuitives : lecture, pause, saut de morceaux et plus encore.

Pour accéder au code source, rendez-vous sur GitHub : github.com/N2630/Music-Bot.
